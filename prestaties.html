the Judging Giraffe

This is judging giraffe. judging giraffe is my first ever programming project and got his name from judging jokes. If you'd shake him after making a joke, he'd decide if the joke was either good, bad or mid.

Gary the Joke Judger

Gary is a lot like judging giraffe considering Gary is his upgrade. He does everything judging giraffe did as well, but there's some differences. For example, Gary will get mad if you make too many bad jokes or if you hold him upside down. But if you hug him, he'll play a song for you.

Reservation System bar-eetcafe de Gouden Leeuw

For this project I got tasked with making a reservation system for a real costumer and so that is what I did. I built a reservation system where a guest can reserve for a specific day at a specific time and can fill in if they're allergic to anything. On the admin side days can be partly or completely closed, reservations can be edited and deleted and you can see a weekly calendar showcasing all of that weeks reservations.

Hefpark platformer

This project was one we made in two weeks time. I made the boss fight which you can see on the image to the side here. The game was made using Javascript and the game library Excaliburjs. The game has four levels which go through the different parts of Hefpark until you eventually reach the end and fight Terry, the boss. The game can be played here: https://flurry19.github.io/CLE4-Game/
